Providers

Registered provider adapters for AI websites (ChatGPT, Claude, DeepSeek, etc.). Each provider is defined as a Python class with @register_adapter in src/proxy/providers/adapters/. Automatically discovered on startup.

Cookies

Authentication profiles with login sessions captured via Playwright browser. Each profile stores cookies and storage state for a specific provider.

API Keys

Virtual API keys in wsk_live_* format. Compatible with any OpenAI client. Set OPENAI_BASE_URL=http://localhost:8000/v1 — the proxy routes through the correct provider adapter based on the key.

Dev Tools

Recorder

Browser-based action recorder for reverse-engineering site APIs. Captures user actions and XHR traffic to help build provider adapters for new websites.

Settings

Proxy configuration, browser engine, and application parameters.
